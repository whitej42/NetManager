Device Management

Manage and configure all your network devices from one location without need for the command line. NetManager allows you to connect to your devices, retrieve device information and send configuration.

Secure Connections

NetManager utilises the Netmiko SSH library to establish secure connections to your devices. Information is polled from each device periodically using the established SSH connection to the network device.

Cloud Backups

Backup your device configuration with the quick and easy cloud backup system. NetManager can store configuration backups for all your network devices which can be easily downloaded using the GUI.

Audit Trail

Keep track of device configuration changes in realtime and mitigate unauthorised configuration changes. Every configuration change made using the NetManager API is tracked and stored for you to view.
