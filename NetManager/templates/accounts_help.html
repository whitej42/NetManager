This section explains how to create, edit and delete access lists

1. Getting Started
  1. Navigate to the 'Device Configuration' page
    See 'Device Configuration'
  2. In the 'Device Configuration' page, navigate to the 'Device Security Features' widget
  3. Click 'Manage Access Lists'
2. Creating Access Lists
  1. Fill out the 'Edit and Create Access List' form (shown below)

  2. Click the 'Push Changes' button to create the new access list
3. Editing Access Lists
3.1 Appending a new line
  1. Using the 'Edit and Create Access List' form
  2. Enter the name of the access list you wish to append a new line to
  3. Enter the desired command for the new line
  4. Click the 'Push Changes' button to add the new line
3.2 Deleting an existing line
  1. Using the 'Edit and Create Access List' form
  2. Enter the name of the access list you wish to delete from
  3. In the command box - Enter 'no' followed by the command of the line you wish to delete
  4. Click the 'Push Changes' button to remove the line
4. Deleting Access Lists
  1. Select the access list you wish to delete from the 'Delete Access List' form dropdown
  2. Click the 'Delete Access List' button

This section explains how to apply and remove access lists from an interface

1. Getting Started
  1. Navigate to the 'Device Configuration' page
    See 'Device Configuration'
  2. In the 'Device Configuration' page, navigate to the 'Device Interfaces' widget
  3. Click the 'More Details' button next to the desired interface
  4. This will then redirect to the 'Interface Configuration' page
2. Applying an Access List
  1. In the interface configuration page, navigate to the 'Access Lists' widget
  2. Select the access list from the dropdown and the required direction
  3. Click the 'Apply Access List' button to continue
3. Removing an Access List
  1. In the interface configuration page, navigate to the 'Access Lists' widget
  2. Select the access list you wish to remove from the interface
  3. Click the 'Remove' button to remove the access list from the interface
